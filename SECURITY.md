# Security Policy

## Supported Versions

Currently, only the `main` branch is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them via email to `gayaladitya9@gmail.com`. 
You should receive a response within 48 hours. If the issue is confirmed, a patch will be developed and deployed as quickly as possible.
